# Task: User Management Service and Types

## Goal
Build the service and types layer of the `user-management` feature: `UserManagementService` that handles all CRUD API calls for users, plus TypeScript types. This is the data layer consumed by the user management UI components.

## Context
The existing React implementation: `projects/application/frontend/app/src/features/user-management/services/` and `types.ts`. The Angular version is an `@Injectable` service using `HttpClient`, returning `Observable<T>` for all operations.

Backend user endpoints (unchanged):
- `GET /users` — query params: `search?: string; page?: number; limit?: number; sortBy?: string; sortOrder?: 'asc' | 'desc'` → `UserListResponse`
- `GET /users/:id` → `User`
- `POST /users` — body: `CreateUserDto` → `User`
- `PUT /users/:id` — body: `UpdateUserDto` → `User`
- `DELETE /users/:id` → `void`

## What to Build

Create `src/features/user-management/` with:

- `types.ts`:
  - `User { id: string; username: string; email: string; firstName: string; lastName: string; roles: string[]; createdAt: Date; }`
  - `CreateUserDto { username: string; email: string; password: string; firstName: string; lastName: string; roles?: string[]; }`
  - `UpdateUserDto` — partial of CreateUserDto minus password
  - `UserListQuery { search?: string; page?: number; limit?: number; sortBy?: string; sortOrder?: 'asc' | 'desc'; }`
  - `UserListResponse { users: User[]; total: number; page: number; limit: number; }`
- `user-management.service.ts` — `@Injectable({ providedIn: 'root' })`:
  - `private http = inject(HttpClient)`
  - `getUsers(query: UserListQuery): Observable<UserListResponse>`
  - `getUser(id: string): Observable<User>`
  - `createUser(dto: CreateUserDto): Observable<User>`
  - `updateUser(id: string, dto: UpdateUserDto): Observable<User>`
  - `deleteUser(id: string): Observable<void>`
- `index.ts` — barrel: `export { UserManagementService } from './user-management.service'; export * from './types';`

## Acceptance Criteria
- [ ] `UserManagementService` is `providedIn: 'root'`
- [ ] All methods return `Observable<T>` (not Promises)
- [ ] `getUsers()` passes query params correctly via `HttpParams`
- [ ] TypeScript interfaces match the backend API contract
- [ ] `ng build` succeeds
- [ ] Type-check passes
