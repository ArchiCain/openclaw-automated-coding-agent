# Task: App Header Component

## Goal
Build the `app-header` feature: the top `<mat-toolbar>` with app title, hamburger menu button (mobile), avatar menu (user profile/logout), and navigation menu. Replaces the React `app-header` feature.

## Context
The existing React implementation: `projects/application/frontend/app/src/features/app-header/app-header.tsx` with sub-components in `avatar-menu/` and `navigation-menu/`. The Angular version uses `<mat-toolbar>`, `<mat-menu>` for the avatar dropdown.

The header needs to:
- Show a hamburger button on mobile that opens the navigation drawer (communicated via `@Output()` or a `LayoutService` event)
- Show app title
- Show avatar `<button mat-icon-button>` with `<mat-icon>account_circle</mat-icon>` that opens a `<mat-menu>` with username, theme toggle, and logout
- The logout calls `AuthService.logout()`
- The theme toggle uses `ThemeToggleComponent` from the `theme` feature

**Important for E2E compatibility**: Logout button must have `aria-label` containing "logout" (current E2E tests use `button[aria-label*="logout" i]`).

## What to Build

Create `src/features/app-header/` with:

- `app-header.component.ts` — standalone `@Component`:
  - `selector: 'app-header'`
  - Imports: `[MatToolbarModule, MatButtonModule, MatIconModule, MatMenuModule, CommonModule, ThemeToggleComponent, AsyncPipe]`
  - `@Output() menuToggled = new EventEmitter<void>()` — emits when hamburger clicked
  - Injects `AuthService` for current user display and logout
  - `onLogout()` — calls `authService.logout()` then navigates to `/login`
- `app-header.component.html` — template:
  - `<mat-toolbar color="primary">`
  - Hamburger: `<button mat-icon-button (click)="menuToggled.emit()" aria-label="Toggle menu"><mat-icon>menu</mat-icon></button>`
  - App title
  - Spacer flex div
  - Avatar button: `<button mat-icon-button [matMenuTriggerFor]="userMenu" aria-label="User menu - logout"><mat-icon>account_circle</mat-icon></button>`
  - `<mat-menu #userMenu>` containing username display, `<app-theme-toggle>`, logout button with `aria-label="logout"`
- `index.ts` — barrel exporting `AppHeaderComponent`

## Acceptance Criteria
- [ ] `AppHeaderComponent` is standalone
- [ ] Hamburger button emits `menuToggled` output
- [ ] Avatar menu shows current username
- [ ] Logout button has `aria-label` containing "logout"
- [ ] Logout calls `AuthService.logout()` and redirects to `/login`
- [ ] `ThemeToggleComponent` is embedded in avatar menu
- [ ] `ng build` succeeds
- [ ] Type-check passes
