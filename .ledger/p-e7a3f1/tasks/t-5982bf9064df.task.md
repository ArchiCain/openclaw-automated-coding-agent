# Task: App Wiring (Routes, Providers, Bootstrap)

## Goal
Wire together the complete Angular application: define all routes in `app.routes.ts`, configure all providers and interceptors in `app.config.ts`, finalize `app.component.ts`, and update `Taskfile.yml` with Angular CLI commands. This is the final integration task that connects all features.

## Context
The scaffold task (t-6373994e32d2) created stub versions of `app.routes.ts`, `app.config.ts`, and `app.component.ts`. All features are now built. This task replaces the stubs with the full wiring.

## What to Build

**`src/app.routes.ts`** — define all application routes:
```typescript
export const routes: Routes = [
  { path: 'login', component: LoginComponent },
  {
    path: '',
    component: AppLayoutComponent,
    canActivate: [authGuard],
    children: [
      { path: '', component: ChatPage, pathMatch: 'full' },
      { path: 'smoke-tests', component: SmokeTestsPage },
      {
        path: 'admin/users',
        canActivate: [permissionGuard('admin')],
        children: [
          { path: '', component: UsersListPage },
          { path: 'new', component: CreateUserPage },
          { path: ':id', component: EditUserPage },
        ]
      },
    ]
  },
  { path: '**', redirectTo: '' }
];
```

**`src/app.config.ts`** — full application config:
- `provideRouter(routes, withComponentInputBinding())`
- `provideHttpClient(withInterceptors([authInterceptor, tokenRefreshInterceptor, activityInterceptor]))`
- `provideAnimationsAsync()`
- `APP_INITIALIZER` that calls `AuthService.checkAuth()` on startup (to restore session and set initial theme via `ThemeService.initialize()`)

**`src/app.component.ts`** — finalize:
- Standalone, imports `[RouterOutlet]`
- No layout here — layout is in `AppLayoutComponent`

**`projects/application/frontend/Taskfile.yml`** — update commands:
- `frontend:local:start` → `ng serve` (or Docker with ng serve)
- `frontend:local:build` → `ng build`
- `frontend:local:test` → `ng test --watch=false`
- `frontend:local:type-check` → `ng build --configuration development` or `tsc --noEmit`
- `frontend:local:lint` → `ng lint` (if eslint configured) or remove

## Acceptance Criteria
- [ ] All 6 routes defined: `/login`, `/`, `/smoke-tests`, `/admin/users`, `/admin/users/new`, `/admin/users/:id`
- [ ] `authGuard` protects all routes except `/login`
- [ ] `permissionGuard('admin')` protects `/admin/*` routes
- [ ] All 3 interceptors registered via `withInterceptors([])`
- [ ] `APP_INITIALIZER` calls `AuthService.checkAuth()` before app renders
- [ ] `ThemeService.initialize()` called with theme from `checkAuth()` response
- [ ] `ng build` produces working application bundle
- [ ] `Taskfile.yml` commands work for Angular CLI
- [ ] Type-check passes
