# Task: Frontend — BackendConfigService and Backend Selector Component

## Goal
Add a `BackendConfigService` and a `BackendSelectorComponent` to the Angular frontend that lets users switch between the NestJS and FastAPI backends at runtime. The selection is persisted in `localStorage` and drives all HTTP calls.

## Context
The Angular frontend is at `projects/application/frontend/app/src/`. It uses standalone components, Angular Material, and `HttpClient` with functional interceptors (see `app.config.ts`).

The `AppHeaderComponent` at `projects/application/frontend/app/src/features/app-header/` is where the toggle will be embedded. The `api-client` feature at `projects/application/frontend/app/src/features/api-client/` provides the WebSocket service — the backend URL change must also affect WebSocket connections.

**Backend URL contract:**
- `nestjs`: empty string `""` — all API calls are relative (same origin, current behavior)
- `fastapi`: `http://api-python.mac-mini` — absolute URL for the FastAPI backend

**LocalStorage key:** `preferred-backend` (values: `"nestjs"` | `"fastapi"`)

This task creates the service and component. The HTTP interceptor that uses this service is a separate task (`t-6220e02b0dda`). The `AppHeaderComponent` integration is also a separate task (`t-d31f65e5c587`).

## What to Build

### `app/src/features/backend-selector/`
A new **shared feature** (no pages, no routes).

**`backend-config.service.ts`:**
```typescript
@Injectable({ providedIn: 'root' })
export class BackendConfigService {
  private readonly STORAGE_KEY = 'preferred-backend';
  readonly selectedBackend$ = new BehaviorSubject<'nestjs' | 'fastapi'>(this.loadFromStorage());

  getBaseUrl(): string  // returns '' for nestjs, 'http://api-python.mac-mini' for fastapi
  setBackend(backend: 'nestjs' | 'fastapi'): void  // updates BehaviorSubject + localStorage
  getSelectedBackend(): 'nestjs' | 'fastapi'  // sync getter
  private loadFromStorage(): 'nestjs' | 'fastapi'  // reads localStorage, defaults to 'nestjs'
}
```
The FastAPI base URL should be read from an environment variable / Angular environment config if possible (`environment.fastapiUrl`), falling back to `'http://api-python.mac-mini'`.

**`backend-selector.component.ts` (standalone):**
A `mat-button-toggle-group` component with two options: `NestJS` and `FastAPI`.
- Reads current selection from `BackendConfigService.selectedBackend$`
- On change, calls `BackendConfigService.setBackend()`
- Uses `MatButtonToggleModule`
- Selector: `app-backend-selector`

**`index.ts`:** Re-exports `BackendConfigService` and `BackendSelectorComponent`.
